What is Linux ? What are its pros and cons ?
Linux is an open-source operating system - the piece of software that is responsible for your computer to work. Because it is open-source, Linux is very secure, and respects user's privacy. Contrary to Windows and Mac OS, Linux does not collect any user data.
On top of that, Linux has many other advantages; for example, it very fast, stable, and allows you to customize it the way you want by changing components like its user interface. Finally, Linux is also free, and can usually be tested using a simple USB stick !
